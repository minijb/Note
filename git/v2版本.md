# git

## 1. 全局配置

**设置用户名**

```shell
git config --global user.name xxx
```

**设置邮箱**

```shell
git config --global user.email xxx
```

**查看控制列表**

```shell
git config --list
```

**初始化**

```shell
git init
```

****

**配置代理**

```shell
git config --global http.proxy 'socks5://127.0.0.1:10808'

git config --global https.proxy 'socks5://127.0.0.1:10808'
```

**删除代理**

```shell
git config --global --unset http.proxy
 
git config --global --unset https.proxy
```

**查看代理**

```shell
git config -l
```

## 2. 基本操作

**初始化**

```shell
git init
```

**查看仓库状态**

```shell
git status
```

****

### 暂存区

**向暂存区添加文件按**

```shell
git add xxx
```

xxx可以为文件名，*则暂存所有的文件

**删除暂存区文件**

```shell
git rm --cached <file>
```

**删除暂存区和工作区的文件**

```shell
git rm <file>
```

**放弃工作区的修改**

```shell
git restore <file>
```

**放弃暂存区的提交**

```shell
git restore --staged <file>
```

**一步到位的提交**

```shell
git commit -a -m "xxx"
```

### 提交

```shell
git commit -m xxx
```

## 3. 日志

查看日志

```shell
git log
```

查看操作历史

```shell
git reflog
```

## 4 .版本回退

**回退版本**

```shell
git reset --hard <标志>
```
