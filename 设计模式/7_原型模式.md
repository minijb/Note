# 原型模式

用原型模式指定创建对象的种类，并通过拷贝这些原型创建新的对象

![image.png](https://s2.loli.net/2022/02/18/1WR2LYvbfFocxtM.png)

- 浅克隆：创建一个新对象，新对象的属性和原来对象完全相同，对于非基本类型属性，仍指向原有属性所指向的对象的内存地址。
- 深克隆：创建一个新对象，属性中引用的其他对象也会被克隆，不再指向原有对象地址。

Java 中的 Object 类提供了浅克隆的 clone() 方法

```java
public abstract class Prototype implements Cloneable {
    /**
     * 类原型，通过Cloneable接口实现浅克隆
     */
    private String name;

    public Prototype(String name) {
        this.name = name;
    }

    public String GetName(){
        return this.name;
    }

    public abstract Prototype Clone();

}

public class ConcretePrototype extends Prototype {

    public ConcretePrototype(String name) {
        super(name);
    }

    @Override
    public Prototype Clone() {
        
        try {
            return (Prototype)super.clone();
        } catch (CloneNotSupportedException e) {
            e.printStackTrace();
            return null;
        }
    }
    
}

public class App {
    public static void main(String[] args) throws Exception {
        ConcretePrototype p1 = new ConcretePrototype("I");
        ConcretePrototype p2 = (ConcretePrototype)p1.Clone();
        System.out.println(p1 == p2);//false
        System.out.println(p2.GetName());//I
    }
}




```
