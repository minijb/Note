## 创建对象

```xml
<bean id="stu" class="org.example.pojo.Student"></bean>
```

- id：创建对象的名称
- class：反射的目录

**在代码中使用对象**

```java
//        创建spring容器，并启动才可以创建对象
        ApplicationContext ac = new ClassPathXmlApplicationContext("applicationContext.xml");
        Student stu = (Student) ac.getBean("stu");
        System.out.println(stu);
```

**给创建的对象赋值**

- 使用setter注入法

注：注入分为简单/引用类型注入

简单类型注入使用value属性，引用类型注入使用ref属性

注：使用setter注入必须提供无参的构造方法，必须提供`setXXX()`方法

```xml
    <bean id="stu" class="org.example.pojo.Student">
        <property name="name" value="张三"/>
        <property name="age" value="21"/>
    </bean>
```

**注入引用类型**

```xml
    <bean id="stu" class="org.example.pojo.Student">
        <property name="name" value="张三"/>
        <property name="age" value="21"/>
        <property name="school" ref="school"/>
    </bean>

    <bean id="school" class="org.example.pojo.School">
        <property name="address" value="清华"/>
        <property name="name" value="随便"/>
    </bean>
```

可以看见通过ref可以调用其他bean进行初始化

- 使用构造方法注入

  - 使用构造方法的参数名称进行注入值

    ```xml
    <bean id="school" class="org.example.pojo.School">
            <constructor-arg name="name" value="清华"/>
            <constructor-arg name="address" value="xxx"/>
        </bean>
    ```

    ```java
        public School(String name, String address) {
            this.name = name;
            this.address = address;
        }
    ```

    

  - 使用构造方法的参数下标注入值

    ```xml
    <bean id="school" class="org.example.pojo.School">
            <constructor-arg index="0" value="清华"/>
            <constructor-arg index="1" value="xxx"/>
        </bean>
    ```

    

  - 使用漠然的构造方法的参数顺序注入值

    ```java
    <bean id="school" class="org.example.pojo.School">
        <constructor-arg  value="清华"/>
        <constructor-arg  value="xxx"/>
    </bean>
    ```



## 创建三层结构

三层结构

![image-20220802003116421](img/image-20220802003116421.png)

dao层

```java
public class StudentMapperImpl implements StudentMapper{

    @Override
    public int insert(Student S) {
        System.out.println("dao==============");
        return 1;
    }
}
```

service层

```java
public class StudentServiceImpl implements StudentService {
    //    必须有数据访问层对象
    private final StudentMapper studentMapper = new StudentMapperImpl();
    @Override
    public int insert(Student s) {
        System.out.println("service==================");
        return studentMapper.insert(s);
    }
}

```

controller

```java
public class StudentController {

    public StudentService studentService = new StudentServiceImpl();

    public  int insert(Student S){
        System.out.println("controller====================");
        studentService.insert(S);
        return 1;
    }
}

```

测试

```java
no input.....

service==================
dao==============
```

通过spring创建这些对象

```xml
    <bean id="sMapper" class="org.example.dao.StudentMapperImpl"/>
    <bean id="sService" class="org.example.service.impl.StudentServiceImpl">
        <property name="studentMapper" ref="sMapper"/>
    </bean>

    <bean id="sController" class="org.example.controller.StudentController">
        <property name="studentService" ref="sService"/>
    </bean>

```

注，在实现类中需要设置setter方法

## 基于注解的IOC

- 创建对象的注解
  - `@Component`可以创建任意对象
  - `@Controller`专门用来创建控制器对象，可以用来接收用户请求，可以返回处理结果给客户端
  - `@Service`专门创建业务逻辑层的对象
  - `@Repository`专门创建数据访问层的对象
- 依赖注入的注解
  - 值类型的注入`@Value`
  - 引用类型注入
    - `@Autowired`使用类型注入值，从bean工厂搜索同源类型的对象进行注入
      - 类可以完全相同，也可以是父子类（注入子类是可以的）
      - 注入的实现类（原本为接口，注入实现类）
    - `@Qualifier`使用名称诸如之，从bean工厂搜索相同名称类型的对象进行注入

```java
@Component
public class School {
    private String name;
    private String address;

    public School() {
    }

    @Override
    public String toString() {
        return "School{" +
                "name='" + name + '\'' +
                ", address='" + address + '\'' +
                '}';
    }


}
```

使用注解就需要添加包扫描

```java
    @Test
    public void testSchool() {
        ApplicationContext ac = new ClassPathXmlApplicationContext("applicationContext.xml");
        School school = (School) ac.getBean("school");
        System.out.println(school);
    }
```

注：注解创建对象的名称为驼峰命名法，也可以指定名称`@Component("xxx")`

### 注入值

**简单值的注入**

```python
    @Value("清华")
    private String name;
    @Value("xxx")
    private String address;

```

**引用注入**

```java
    @Autowired
    private Student student;
```

```java
@Component("stu")
public class Student {    


@Autowired
@Qualifier("stu")
private Student student;
```

如果父类和子类都进入了bean，那么autowird会注入哪一个？

这时候有两个对象可以注入，此时，按照名称进行二次筛选，选中与被注入对象相同名称的对象进行注入。但是我们可以通过名称来自己选择

### 修改三层框架

```java
@Controller
public class StudentController {
    @Autowired
    public StudentService studentService;

    public  int insert(Student S){
        System.out.println("controller====================");
        studentService.insert(S);
        return 1;
    }

    public void setStudentService(StudentService studentService) {
        this.studentService = studentService;
    }

    @Repository
    public class StudentMapperImpl implements StudentMapper{

        @Override
        public int insert(Student S) {
            System.out.println("dao==============");
            return 1;
        }
    }


    @Service
    public class StudentServiceImpl implements StudentService {
        //    必须有数据访问层对象
        @Autowired
        private StudentMapper studentMapper;
        @Override
        public int insert(Student s) {
            System.out.println("service==================");
            return studentMapper.insert(s);
        }

        public void setStudentMapper(StudentMapper studentMapper) {
            this.studentMapper = studentMapper;
        }
    }

```

## 包扫描

- 直接扫描根目录

- 单个包扫描

  ```xml
  <context:component-scan base-package="org.example.pojo"/>
  ```

- 多个包扫描

  ```xml
   <context:component-scan base-package="org.example.pojo,org.example.controller"/>
  ```

### 指定多个spring 配置文件

- 按层---controller,service,dao
- 按功能拆---users,studentxxxx

### 导入spring配置文件

- 单个文件导入

  ```xml
  <import resource="applicationContext.xml"/>
  ```

- 批量导入

  ```xml
  <import resource="applicationContext_*.xml"/>
  ```

### xml中引用类型的自动注入

```xml
    <bean id="stu" class="org.example.pojo.Student" autowire="byType/byName">
        <property name="name" value="张三"/>
        <property name="age" value="21"/>
        <!--<property name="school" ref="school"/>-->
    </bean>
```
