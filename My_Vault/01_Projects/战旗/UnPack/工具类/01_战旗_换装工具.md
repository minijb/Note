## 1. Spine 动画结构

1. SkeletonAnimation ---  创建对应实体之后的动画对象
2. SkeletonDataAsset ---  ScriptObj -- 数据资源 -- 可以使用他生成对应的预制体或者 gameobject [doc](https://zh.esotericsoftware.com/spine-unity-main-components)
3. SkeletonData      --- 数据信息  存放谷歌层次结构，槽位等
4. Skeleton          --- 实例对象

[SkeletonAnimation](https://zh.esotericsoftware.com/spine-unity-main-components#SkeletonAnimation%E7%BB%84%E4%BB%B6) 组件提供了 `SkeletonAnimation.Skeleton` 属性以便访问底层的 [Skeleton](https://zh.esotericsoftware.com/spine-unity-main-components#Skeleton). Skeleton存储了对skeleton data资产的引用, 而skeleton data又引用了数个atlas资产.

因此一般对于 slot bones 的寻找都是使用 Skeleton 对象。


## 2. 资源结构

spine 使用 图集 atlas 作为资源
atlas ： 在一个或多个texture页中存储关于texture区域的信息.
page ： atlas内含的texture的设置.
region ： atlas页上的texture区域.
[Attachment](https://zh.esotericsoftware.com/spine-api-reference#Attachment) ： 附件  用于附着在xx 上的东西
[RegionAttachment](https://zh.esotericsoftware.com/spine-api-reference#RegionAttachment) ： 一个用于显示texture四边形的附件


### 3. 如何进行组建替换

**资源准备**

1. Spine资源文件中还有对应的 Slot 
2. texture 文件 --- png
3. 准备一个 material ---- 
	1. 注意实现将 `Straight Alpha Texture` 的选项勾上。
	2. 否则会出现虚影

### 4. 组装过程

1. 获得基底的 Skeleton
2. 通过 png 生成 Spine -- Atlas、Region
3. 通过 Region 生成 Attachment -- `RegionAttachment`
4. 替换对应的 RegionAttachment
	1. 注意：将偏移以及旋转应用到复制到新的 Attachment 中

```c#
public Atlas GetSpineAtlas(BustPart_Asset asset)
{
	if (asset.atlas != null) return asset.atlas;
	Texture tex = GetBustPart_Texture(asset);

	AtlasPage page = new AtlasPage();
	page.width = tex.width;
	page.height = tex.height;
	page.format = Format.RGBA8888;
	page.magFilter = TextureFilter.Linear;
	page.minFilter = TextureFilter.Linear;
	page.uWrap = TextureWrap.ClampToEdge;
	page.vWrap = TextureWrap.ClampToEdge;
	page.name = asset.AssetPath;
	page.pma = true;
	Material m = GetBustPart_Material(asset, tex);
	page.rendererObject = m;

	AtlasRegion region = new AtlasRegion();
	region.page = page;
	region.name = asset.AssetPath;
	region.index = 3;
	region.originalHeight = tex.height;
	region.originalWidth = tex.width;
	region.rotate = false;
	region.offsetX = 0;
	region.offsetY = 0;
	region.packedHeight = tex.height;
	region.packedWidth = tex.width;
	region.x = 2;
	region.y = 2;
	region.u = 0;
	region.u2 = 1;
	region.v = 1;
	region.v2 = 0;



	List<AtlasPage> atlasPages = new() { page };
	List<AtlasRegion> atlasRegions = new() { region };
	asset.atlas = new Atlas(atlasPages, atlasRegions);
	return asset.atlas;
}

public RegionAttachment GetSpineRegionAttachment(BustPart_Asset asset, RegionAttachment target_attachment)
{
	if (asset.regionAttachment != null) return asset.regionAttachment;

	Atlas atlas = GetSpineAtlas(asset);
	AtlasRegion region = atlas.Regions[0];
	RegionAttachment attachment = region.ToRegionAttachment(region.name);
	attachment.X = target_attachment.X;
	attachment.Y = target_attachment.Y;
	attachment.Rotation = target_attachment.Rotation;
	attachment.ScaleX = target_attachment.ScaleX;
	attachment.ScaleY = target_attachment.ScaleY;
	attachment.UpdateRegion();

	asset.regionAttachment = attachment;
	return attachment;
}
```
