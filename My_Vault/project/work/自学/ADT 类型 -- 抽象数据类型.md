# ADT 类型   抽象数据类型

1. **封装**：将数据和操作数据的函数封装在一起，隐藏了数据结构的内部实现。
2. **抽象**：只关注于操作，而不关心数据的具体存储方式。
3. **接口**：定义了ADT可以执行的操作，即公开的方法。
4. **实现**：具体实现细节对外部用户隐藏，仅通过接口进行交互。
