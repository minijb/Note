# Nodejs

使用 nvm 管理多版本

```sh
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.5/install.sh | bash
source ~/.bashrc  # 或 ~/.zshrc


# 使用 nvm 管理 node 版本
nvm install node    #最新稳定版本
nvm use xxx         #使用指定版本
```
