分为数组本分和哈希部分

- 数组部分 从 1 开始的整数索引， 需要连续
- 哈希部分， 就是剩下的， nil 不能作为 key


**rehash**


如果空间不够用分配更多空间，如果空间利用率小， 则缩减内润。

- 计算 array part 的key 数量
- 计算 hash part 的 key 数量

- 计算一个新的 array part 部分需要分配的内存大小
