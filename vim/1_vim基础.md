# VIM

## 1.基础

### 1.1常用操作

#### 移动

- `h j k l`

> word和WORD
>
> - word是由空白字符分割的**字母，数字和下划线的组合**
> - WORD由空白字符分割的**非空字符串**

| 按键 | 行为                     |
| ---- | ------------------------ |
| w    | 狭义单词移动             |
| e    | 向后到最近的狭义单词结尾 |
| W    | 广义                     |
| E    | 广义                     |
| b    | 向前移动到狭义单词的开头 |
| B    | 广义                     |

| 命令 | 行为         |
| ---- | ------------ |
| {    | 向前一个段落 |
| }    | 向后         |

#### 简单编辑

- 修改命令c

在删除一部分后自动进入插入模式

修改命令可以和移动命令结合达到修改部分代码的功能

如cw---删除一个单词，c3e----删除3个e的距离。。。。。

- 删除命令d

删除一部分代码，不进入插入模式

- cc，dd整行命令

## 2. 工作区

- 缓冲区代表文件，通过缓冲区可以切换文件
- 窗口：在同一个屏幕上显示多个文件
- 标签：窗口的分组

#### 缓冲区

通过`:ls`来查看缓冲区

`:e xxxx`来打开其他文件的缓冲区

缓冲区的跳转`:b xxx` xxx可以是序号也可以是文件的简写，但是不能匹配多个，如果匹配可以使用tab补全

> 特殊的
>
> `:bn` `bp`（next ,previous）

#### 窗口

可以同时显示多个缓冲区

`:split` `:vsplit` 水平/垂直分割窗口

后面可以加文件名，如果没有则显示当前的缓冲区

简化:`:sp` `:vs`

- 在窗口之间移动

`ctrl+w`后加方向键或者`ctrl+hjkl`

- 关闭窗口

`ctrl+w  q`关闭当前窗口

`:q`退出窗口并卸载缓冲区

`:bd`删除缓冲区并关闭当前窗口

`:only/on`或者`ctrl+w o`关闭除当前窗口外所有窗口

- 移动窗口

`ctrl+w  HJKL`将窗口移动到最xx边

`ctrl+w r`将窗口内容向右(下)移动  R相反

`ctrl+w x`和下一个窗口交换内容

- 窗口大小

`ctrl+w =`窗口等宽高

`ctrl+w +/-`调整高度

`ctrl+w >/<`调整宽度

#### 标签页

新建tab`:tabnew`

- 切换tab

`gt`  `:tabnext`切换到下一个标签

`gT`  `:tabprevious`切换到前一个标签

## 3. 文本浏览

#### 行内移动

- `h l`
- `t+[字符]`T反向搜索并将光标移动到该字符前
- f，F移动到该字符上
- _移动到行首,$移动到行尾
- gg文件首页，G文件末尾

#### 搜索

`/`正向搜索，`?`反向搜索

`:noh`取消高亮

#### 利用文本对象

文本对象通常包括

- w,W
- s  句子
- p  段落
- t  标签（html）
- `'  "  ()  []  {}`

`i  a` 

- i 为内对象 如`di)`删除括号内的东西
- a为外对象  c2aw  删除后两个单词

## 3. 寄存器

- 查看寄存器

`:reg`可以在空格后加上寄存器的名称查看寄存器里的值
