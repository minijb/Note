# 逻辑回归

经典的二分类方法

一般都是先用简单的算法，能用简单的还是用简单的

逻辑回归的决策边界：可以是非线性的

### Sigmoid函数

$$
g(z)={1 \over {1+e^{-z}}}
$$

函数图像

```python
from matplotlib import pyplot as plt
import numpy as np
x = np.arange(-10, 10, 0.1)
y = 1/(1+np.exp(-x))

ax = plt.gca()
ax.spines['right'].set_color('none')  # 将右边 边沿线颜色设置为空 其实就相当于抹掉这条边
ax.spines['top'].set_color('none')
ax.xaxis.set_ticks_position('bottom')
ax.yaxis.set_ticks_position('left')

## 设置中心的为（0，0）的坐标轴
ax.spines['bottom'].set_position(('data', 0))  # 指定 data 设置的bottom(也就是指定的x轴)绑定到y轴的0这个点上
ax.spines['left'].set_position(('data', 0))
plt.plot(x, y)
plt.show()
```

他将输入得到一个概率
$$
h_\theta(x)=g(\theta^Tx)={1 \over {1+e^{-\theta^Tx}}}
$$
执行分类任务

$P(y=1|x;\theta)==g(\theta^Tx)$

$P(y=0|x;\theta)=h_\theta(x)=1-g(\theta^Tx)$

整合

$P(y|x;\theta)=(h_\theta(x))^y(h_\theta(x))^{(1-y)}$

对数似然函数
$$
l(\theta)=logL(L(\theta))=\sum_{i=1}^m(y_i\log{h_\theta(x_i)}+(1-y_i)\log{(1-h_\theta(x_i))})
$$
因为这是一个上升问题，此时我们引入梯度下降$J(\theta)=-{1 \over m}l(\theta)$



求导
$$
{\partial \over \partial \theta_j}J(\theta)=-{1 \over m}\sum_{i=1}^m(y_j-g(\theta^Tx_i))x_i^j
$$
可以发现主要部分和梯度下降类似
