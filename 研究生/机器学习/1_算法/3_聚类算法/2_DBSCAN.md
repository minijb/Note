# DBSCAN

- 核心对象：若某个点的密度达到算法设定的阈值则其为核心点(即r临近域内点的数量不小于minPts)
- $\in$-临近的距离阈值：设定半径r
- 直接密度可达：若某点p在点q的r领域内，且q是核心点则p-q直接密度可达
- 密度可达：若有一个点的序列$q_0,q_1,....q_k$，对任意$q_i-q_{i-1}$是直接密度可达的，则称$q_0$到$q_k$是密度可达，实际上就是直接密度可达的传播

## 1. 基本概念

- 密度相连
- 边界点：属于某一个类的非核心点，不能发展下线了
- 直接密度可达
- 噪声点：不属于任何一个类簇的点，从任何一个核心点出发都密度不可达



- 核心对象
- 边界点
- 离群点

## 2. 工作流程

- D:数据集
- $\in$指定半径
- MinPts：密度阈值



- 标记对象为unvisited
- 选择一个unvisited的对象p
- 标记其为visisted
- 如果在$\in$领域内有MinPts个对象
  - 创建一个蔟，并把p放入
  - 便利领域内的点
    - 如果unvisited标记为visited
    - 进行检测，并判断是否可以加入蔟
  - 直到都不是核心对象
- 重复之前过程 知道所有p都是visited

## 3. 参数选择

- 半径$\in$可以根据K距离来设定：找突变点

K距离：给定数据集$P=\lbrace p(i);i=0,1,2,...n \rbrace$,计算P(i)到集合D的子集S中之间的距离，按照距离从小到大排序d(K)就是k距离

- Minpts:k距离中的k值，一般取的小一些



## 4. 优劣势

优势

- 不需要指定蔟的个数
- 可以发现任意密度的蔟
- 擅长找离群点
- 两个参数够用

劣势

- 高纬度数据有些困难（可以降维）
- 参数难以选择
- sklearn中效率很慢
