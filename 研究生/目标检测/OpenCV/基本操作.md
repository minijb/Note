# Opencv的基本操作

读取图像

```python
cv2.imread(url)
```

注格式为GBR

展示图片

```python
img = cv2.imread("./img/000064.jpg")
cv2.imshow("img",img)
cv2.waitKey(0)
```

查看图片的形状

```python
img.shape
```

**在读取过程中读取不同格式的图像**

- `cv2.IMREAD_COLOR`
- `cv2.IMREAD_GRAYSCALE`

```python
path = "./img/000064.jpg"
img = cv2.imread(path,cv2.IMREAD_GRAYSCALE)
cv2.imshow("img",img)
cv2.waitKey(0)
img.shape#(500, 496)
```

可以看到就是单通道的一张图片

**图像保存**

```python
cv2.imwrite('cat.jpg',img)
```

## 视频读取

- `cv2.VideoCapture`可以捕获摄像头，用数字来控制不同的设备
- 如果是视频文件直接指定路径

```python
vc = cv2.VideoCapture("img/SampleVideo_1280x720_1mb.mp4")
if vc.isOpened():
    open,frame = vc.read()
else:
    open = False
    
while open:
    ret,frame = vc.read()
    if frame is None:
        break
    if ret == True:
        gray = cv2.cvtColor(frame,cv2.COLOR_BGR2GRAY)
        cv2.imshow('result',gray)
        if cv2.waitKey(10) &0xFF ==27:
            break
vc.release()
cv2.destroyAllWindows()
```
