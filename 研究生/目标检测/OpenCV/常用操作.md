## 创建空白/其他颜色的图像

```python
img = np.zeros((512,512,3),np.uint8)
show(img)
```



## 绘制图形/文字

```python
cv2.rectangle(img,(0,0),(200,200),(0,255,0),3)
show(img)
font = cv2.FONT_HERSHEY_PLAIN
cv2.putText(img,"OPENCV",(10,500),font,1,(255,255,255),1,cv2.LINE_AA)
show(img)
```

## 图像的平移

`cv2.warpAffine(img,M,dsize,borderValue)`

borderValue---在边界不变的情况下使用的值;缺省值是0。也就是

```py
img = cv2.imread("img/000064.jpg")
m = np.array([
    [1,0,100],
    [0,1,100]
],dtype=np.float32)
img = cv2.warpAffine(img,m,dsize=(500, 496),borderValue=(128,128,128))
show(img)
```

## 将图像的周围加上灰色

```python
img = cv2.imread("img/000064.jpg")
width , height = img.shape[:2]

new = np.zeros((800,800,3),dtype=np.uint8)
new = new + 128
new[400-width//2:400+width//2,400-height//2:400+height//2,:] = img
show(new)
```
