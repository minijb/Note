# 迭代器

## 1. 可迭代对象

list , dict . str , set , tuple

## 2. 迭代器

由`__iter__   __next__`的方法的对象就是迭代器

迭代器可以一个一个的取出对象，节约内存

> for in 就是将list等可迭代对象转化为迭代器
>
> `__iter__`将可迭代对象转化为迭代器
>
> `__next__`如果迭代完成指定的次数，我们可以添加一个终止条件来引发错误

## 3. 生成器

构造生成器的方式

- 生成器函数
- 生成器表达式
- 内置函数

#### 生成器函数

```python
def func():
    print(11)
    yield xxx
ret = func()#产生一个生成器对象
```

只要由yield语句该函数就是生成器函数

生成器函数可以`next([生成器对象])`

next和yield一一对应每次next可以接受一个yield的值

yield只会暂停函数不会结束函数

- `yield from [可迭代对象]`

在函数中可以直接迭代一个可迭代对象
