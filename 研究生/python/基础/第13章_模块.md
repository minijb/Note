# 模块

## 1. 导入模块

- `import A`

开辟一个内存空间，通过A.xxx调用A中的对象

- `from A import B`

相当与将A模块的全局空间中将B的对应关系复制到本文件中

优势

- 使用方便
- 节约内存

缺点

- 容易与当前文件产生覆盖(同名)

```python
name = 'hello'


def change():
    global name

    name = 'world'


from new import name
from new import change

print(name)#hello
change()
print(name)#hello
```

**因为from import是将function的对象复制到test中，所以执行change其实是再new中执行，不是修改本文件中的，而本文件的name之前已经复制好 不会修改**

```python
import new

new.change()
print(new.name)# world
```



- `from xxx import *`

少使用，但是在模块文件中可以使用`__all__=[xx,xx]`可以控制*的取值范围，内容为对象的名称的字符串形式



- `__name__`

第一个执行的文件会返回`__main__`

如果不是第一个文件返回模块名
