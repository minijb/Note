字符串发消息转换

upper----大写

lower---小写

titile----单词首字母大写

**在字符串中使用变量**

```python
first_name = "ada"
second_name = "love me"
full_name = f"{first_name} {second_name}"
```

字符串前的f说明字符串是format格式

**删除多余的空白**

`string.rstrip()`

比如:hello world 会将左边最后的空格删除

同样`lstrip()`会将右边所有的空格删除

**如果数字较大可以使用下划线进行分割**

`400_000_000`不会影响原意

**使用，可以给多个变量赋值**

```python
x,y,z = 1,2,3
```

> `strip`类函数可以带有参数表明要删去的内容

python中没有常量，但是可以使用全部大写来标明这个是常量

### 类型转换

- `str()`将内容转化为字符串

常用于将数字类型加入到字符串

- `int()`转换为整型
- `float()`转换为浮点型
- `bool()`将内容转化为布尔类型

### 整型的不同进制

整型的默认进制时十进制

- 二进制

`bin(15)#0b1111`

`int("1111",2)`来将二进制转化为十进制
